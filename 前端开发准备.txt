1.安装nodejs
安装目录：C:\Program Files\nodejs
检查Node.js版本 node --version
安装npm
cd C:\Program Files\nodejs
git clone --recursive git://github.com/isaacs/npm.git 新版本nodejs不需要安装 升级 npm install npm -g
安装express  npm install express -gf  安装express命令工具 npm install -g express-generator
在C:\Users\Administrator\AppData\Roaming\npm\node_modules目录 会有express
卸载模块 npm uninstall express
更新模块 npm update express
搜索模块   npm search express
使用express创建一个工程:
express helloworld
npm install
npm start
